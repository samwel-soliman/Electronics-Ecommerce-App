# Electronics E-Commerce Flutter App 🛒

**Cross-Platform E-Commerce Mobile Application**

Associated with **Arab Academy for Science, Technology and Maritime Transport**  
Developed using **Flutter** and **Dart**.

## Project Overview

Electronics E-Commerce Flutter App is a cross-platform mobile shopping application designed to provide a modern and smooth shopping experience for electronic products.

The application focuses on user-friendly navigation, responsive UI, product browsing, authentication, cart management, and dynamic data handling.

It simulates a production-level mobile commerce experience with a strong focus on performance, usability, and scalability.

## Key Features

- Product listing with detailed information
- Product price, ratings, and images
- Product details screen
- Cart management
- Smart search and filtering system
- Dark and light mode
- Secure authentication with login and signup
- Smooth checkout flow
- Order tracking simulation
- Responsive and user-friendly UI

## Technologies Used

- Flutter
- Dart
- Firebase Authentication
- REST APIs
- Riverpod State Management
- SQLite Local Storage
